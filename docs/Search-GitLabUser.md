# Search-GitLabUser

## SYNOPSIS
Retrives a GitLab user.

## SYNTAX

```
Search-GitLabUser [-User] <String>
```

## DESCRIPTION
Retrives a GitLab user.

## EXAMPLES

### -------------------------- EXAMPLE 1 --------------------------
```
Search-GitLabUser ngetchell
```

name               : Nicholas Getchell
username           : ngetchell
id                 : 2
state              : active
is_admin           : true
